# Harmoniq – Concert Management System 🎵

## Overview
Harmoniq is a web-based Concert Management System that allows organizers to create and manage concerts while attendees can browse available concerts and book tickets. The platform simplifies the process of organizing music events by providing a structured system for event creation, management, and participation.

This project was developed as part of a B.Tech semester project to demonstrate full-stack web application development using modern technologies.

---

## Features

### User Management
- User registration
- User login
- Role-based access control (Organizer / Attendee)

### Organizer Features
- Create new concerts
- Add concert details (title, description, date, location)
- View and manage concerts created by the organizer

### Attendee Features
- Browse available concerts
- View event details
- RSVP or book tickets for events

### Ticket System
- QR code-based ticket generation
- Secure entry verification
- Prevention of duplicate ticket usage
